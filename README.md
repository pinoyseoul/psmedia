# PinoySeoul Media Enterprise Official Blogger Theme Template

This repository contains the official Blogger theme template used to power the company website of PinoySeoul Media Enterprise. Originally developed by SoraTemplates, this theme has been customized and is now maintained by PinoySeoul Media Enterprise, with significant contributions from founder Nash Ang. It is designed to provide a robust and visually appealing online presence.

## Live Preview

A live version of this theme is viewable at [media.pinoyseoul.com](https://media.pinoyseoul.com).

## Features

*   **Customizable Layout:** Easily adapt the theme to various content needs.
*   **Responsive Design:** Ensures optimal viewing experience across a wide range of devices, from desktops to mobile phones.
*   **Integrated with Blogger Platform:** Leverages Blogger's powerful publishing tools for seamless content management.

## Getting Started

To use this theme, clone the repository and upload the `theme.xml` file to your Blogger account.

```bash
git clone https://github.com/pinoyseoul/psmedia.git
```
